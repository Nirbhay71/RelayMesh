F1: 
    Frontend
    s-1 : Click on create new Contact
    s-2 : Enter contact's name and email
    s-3 : Click on save button.
    Backend
    s-4 : Check user with provided email exists in DB.
    s-5 : Create new Contacts entry (id(auto generate), mine(Foreign Key to User(id) will get from toke no db call needed), userId(no db call needed just found in s-4), userName, userEmail)
    s-6 : create a room with name (mine:userId)

Database Schema
    Initial :- 
    User {id, name, email, authType, refreshToken, password(Hashed password), ContactList[{userId, userName, userEmail, chatHistory{message, timestamp, senderId}}]}

    1NF :- 
    User {id(Primary Key), name, email, authType, avatar, refreshToken, password(Hashed password)}
    Contacts {id(Primary Key), mine(Foreign Key to User(id)), userId, userName, userEmail}
    ChatHistory {id(auto generated as Primary Key), roomId(Foreign Key to Contacts(id)), message, timestamp, senderId}
    
    Indexing :-
        User(id) (For finding the avatar)
        Contacts(mine, userId)  (For finding the roomId for ChatHistory)
        ChatHistory(roomId) (For finding the ChatHistory entry in DB)
